RobotTask-4-3. Python Tutorials. Actor Robot. StepInDev Platform. GUI Application. 
The robot's field is 5 cells long and 5 cells wide. It has blue walls around the perimeter. 
The robot's starting position is marked with a blue square. 
The robot's final position is marked with a red house. 
The task is to move the robot from its starting position to its final position. 
Along the way, color in the cells marked with a small orange square (see figures). 
Solve the problem for all robot environments (write a common code).
The solution of the task is located in the file "Task-4-3.py".



RobotTask-4-3. Занятия по Python. Исполнитель Робот. Платформа StepInDev. Приложение с графическим интерфейсом пользователя.
Первая обстановка робота имеет вид лабиринта с размерами: длина 5 клеток, ширина 5 клеток. Оно имеет синие стены по периметру. 
Синим квадратом помечено начальное положение робота. 
Конечное положение робота помечено красным домиком. 
Требуется переместить исполнителя из начального положения в конечное. 
По пути следования закрасить клетки помеченные малым оранжевым кватратом. (см. рис.). 
Решить задачу для всех обстановок робота (написать общий код).
